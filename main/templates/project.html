Overview: This project aimed to explore, clean, and visualize customer transaction data from an e-commerce platform to better understand purchasing behavior, segment customers, and support strategic decision-making. I created an interactive dashboard that brings together multiple layers of analysis into a cohesive, intuitive visual experience. Objectives: To identify key sales trends across time, products, and customer segments To perform customer segmentation using RFM (Recency, Frequency, Monetary) analysis To design a visual tool that allows business stakeholders to quickly extract actionable insights without diving into raw data
